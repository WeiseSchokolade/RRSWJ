# RRSWJ
RRSWJ renders stuff with java

## Usage
#### Maven
```xml
<repository>
  <id>jitpack.io</id>
  <url>https://www.jitpack.io</url>
</repository>
<dependency>
  <groupId>com.github.WeiseSchokolade</groupId>
  <artifactId>RRSWJ</artifactId>
  <version>master-SNAPSHOT</version>
</dependency>
```
